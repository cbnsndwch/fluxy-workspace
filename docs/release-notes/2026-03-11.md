# 2026-03-11 — WebKit Browser Speech-to-Text

> **Commit:** `52f1536` (2026-03-11)
> **Author:** Bruno Bertapeli

---

## Overview

Speech-to-text in the chat input bar now works on Safari and other WebKit-based browsers. Previously, STT relied on the Web Speech API which is only available in Chrome/Edge. This release adds a WebKit-compatible code path.

---

## New Features

### WebKit STT Support

The `useSpeechRecognition` hook was updated to detect the browser's STT API:
- Chrome/Edge: uses `window.SpeechRecognition`
- Safari/WebKit: uses `window.webkitSpeechRecognition`

This means voice input in the chat bubble now works on **iOS Safari**, **macOS Safari**, and any other WebKit browser — which covers iPhone and iPad users accessing Fluxy as a PWA.
