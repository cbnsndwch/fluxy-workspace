## Fixes

### Self-Update Trigger Path Fixed

The `fluxy self-update` flow works by writing a `.update` trigger file that a file watcher picks up to pull and restart. After the workspace restructure, the path Sebastian was writing the trigger file to had drifted from where the watcher was watching.

The `cli/commands/update.ts` was patched to write the trigger file at the correct resolved path. Additionally, the `sudo` dependency was removed — the update no longer requires elevated privileges.
