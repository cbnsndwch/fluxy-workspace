## CRON Issue Worker ‚Äî #34 (Docs deep linking)

  - Auto-select effect now navigates to first file's URL (replace) instead of just setting state
  - Home button, create, and delete all call `navigate('/docs')` or `navigate('/docs/<slug>')`
