## 13:45 — Built: Version History app (git-viewer)

Diego asked for a lightweight read-only git viewer. Registered in App Ideas canvas (id: 34) at start, tracked through "doing" → "built".

### What was built
- **Backend** `backend/apps/git-viewer/index.ts` — 6 routes using `child_process.execSync` + git CLI:
  - `GET /api/git/status` — working tree status (staged/unstaged/untracked, branch, ahead/behind)
  - `GET /api/git/log?skip=&limit=` — paginated commit log
  - `GET /api/git/branches` — local + remote branches
  - `GET /api/git/worktrees` — worktree list
  - `GET /api/git/commit/:sha` — commit detail + unified diff (capped 300KB)
  - `GET /api/git/uncommitted` — staged + unstaged diff
- **Frontend** `client/src/components/GitViewer/GitViewerPage.tsx` — split panel layout:
  - Left: tabbed Log / Branches / Worktrees (320px)
  - Right: commit detail with syntax-colored diff viewer
  - Uncommitted changes row at top of log when dirty
  - Header: clean/dirty badge + refresh button
- **Registry**: `History` icon, `lime` color theme, workspace section, path `/git-viewer`
- **Route**: `git-viewer` in router.tsx
- Linter auto-improved field naming and component polish
